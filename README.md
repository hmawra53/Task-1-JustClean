# JustClean - Task 1

A responsive landing page for **JustClean**, an online maid booking service, developed as **Task 1** for the Full Stack Web Development Internship.

## Features
- Responsive navigation bar with dropdown menu
- Modern hero section with CTA button
- Feature cards with Font Awesome icons
- Responsive layout for desktop and mobile
- Clean footer with quick links and contact details
- Built using HTML5 and CSS3

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Font Awesome
